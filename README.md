# Saluna IT — Web (Next.js + Tailwind)

## Requisitos
- Node.js 18+ (recomendado 20 LTS)

## Arranque local
```bash
npm install
npm run dev
```

Abrir: http://localhost:3000

## Deploy en Vercel
1) Sube este repo a GitHub
2) En Vercel: New Project → Import Git Repository → Deploy

## Dominio
En Vercel: Project → Settings → Domains → añade `salunait.es` y configura los DNS que te indique.
